## ğŸ’» Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need: To clone or fork and run it in a browser

### Setup

Clone this repository to your desired folder:

Example commands:

```sh
  cd my-folder
  git clone https://github.com/migo1/space-traveler.git
```

### Install

- Run ` cd space-traveler `
- Run ` npm install `

### Usage

To run the project, execute the following command:

- Run `npm start`

### Run tests

To run tests, run the following command:

- to test validation errors `npx hint .`

- to test CSS linter errors `npx stylelint "**/*.{css,scss}"`

- to test JavaScript linter errors  `npx eslint .`

- to run the jest tests `npx test`
